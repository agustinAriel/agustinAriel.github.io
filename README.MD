Este proyecto es una página web básica desarollada como parte de un curso de Front-End. La página está estructurada con HTML sémantico y utiliza las etiquetas <header>,<main> y <footer> para organizar el contenido. El objetivo es aprender a crear la estructura básica de una página web y prepararla para futuras mejoras con CSS y JavaScript.
Este proyecto consiste en la creación de una página web dedicada a promocionar y destacar la riqueza vitivinicola de la provincia de Mendoza, enfocándose en la singularidad de cada uno de sus departamentos. La idea es dar a conocer el vino mas representativo o emblemstico de cada departamento.w
